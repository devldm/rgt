# 알지티 assignment

파이지 라우팅

```tsx
/interactive // interactive web page

/simulation // robot simulation

/dashboard // dashboard
```

## 기술 스탁

- Typescript
- Next.js
- tailwindcss
- recharts (Charts / Visualisation)
- pathfinding.js (Robot simulation)

## How to run locally

1. Unzip the folder
2. Install dependencies

```bash
npm install
```

1. Run the project

```bash
npm run dev
```

1. Navigate to [localhost](http://localhost) in browser

```bash
http:///localhost:3000
```
